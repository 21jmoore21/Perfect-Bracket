The Perfect Bracket Simulator: How The Stats Are Processed

Summary

All stats for this website are from ESPN's college basketball statistics. After retrieving all of the stats, I used python to clean up of the data and get it into a format that fits my use case. Once the data was cleaned up, I knew I would have to scale the stats so that metrics like points per game wouldn't have a much larger impact on the combination than other stats like assists per game. In order to even the playing field, I scaled all of the data so that for every stat, the team with the highest value for that stat was set to 1, the team with the lowest value was set to 0 and all other teams are linearly scaled between the two. For example, if Kansas averaged the most ppg out of all teams in the tournament field, their new ppg value would be scaled to 1.

For stats such as ppg, if that stat has a higher weight assigned to it, the teams that averaged a high ppg will be predicted to perform better. For stats such as turnovers, where it is undesirable to have a lot, the data was inverted to ensure that the team who averaged the least amount of turnovers had their TOV value scaled to 1. This means, if the TOV weight is set higher, the teams that took care of the ball will be predicted to perform better.

Once you have the combination of weights set that you believe will predict that perfect bracket and hit run, the score for each team will be displayed by the team name in the first round. This is helpful in seeing how close two teams are predicted to be, as well as understanding which teams have good stats in some areas and others don't. The team's score is not displayed in any rounds other than the first, but the score does not change round per round. If you wish to make slight adjustments to a stat to see how that affects the teams score, simply click on the slider for that stat and use the left/right arrows on your keyboard to make changes in weight by increments of 1 or click on the current weight and type your desired value.

Formulas

  • Offensive Rating [Off Rtg] = 100 * ( Points / Poss )
  • Defensive Rating [Def Rtg] = 100 * ( Opp Points / Poss )
  • Effective Field Goal Percentage [eFG%] = 100 * ( FGM + 0.5*3PM ) / FGA
  • Three Point Percentage [3pt%] = 100 * ( 3PM / 3PA )
  • Offensive Rebounding Percentage [OReb%] = 100 * OReb / ( OReb + Opp DReb )
  • Turnover Percentage [TOV%] = 100 * TOV / ( FGA + 0.44*FTA + TOV )
  • Free Throw Attempt Rate [FTA%] = 100 * ( FTA / FGA )
  • Free Throw Percentage [FT%] = 100 * ( FTM / FTA )
  • Opponent Effective Field Goal Percentage [Opp eFG%] = 100 * ( Opp FGM + 0.5*Opp 3PM ) / Opp FGA
  • Opponent Three Point Percentage [Opp 3pt%] = 100 * ( Opp 3PM / Opp 3PA )
  • Opponent Offensive Rebounding Percentage [Opp OReb%] = 100 * Opp OReb / ( Opp OReb + DReb )
  • Opponent Turnover Percentage [Opp TOV%] = 100 * Opp TOV / ( Opp FGA + 0.44*Opp FTA + Opp TOV )
  • Opponent Free Throw Attempt Rate [Opp FTA%] = 100 * ( Opp FTA / Opp FGA )
  • Possessions [Poss] = 0.5* ( (FGA + 0.4*FTA - 1.07/100*OReb% * ( FGA - FGM ) + TOV) + (Opp FGA + 0.4*Opp FTA - 1.07/100*Opp OReb% * ( Opp FGA - Opp FGM ) + Opp TOV) )
  • Seed In March Madness [Seed] = Seed Selected By Selection Committee
